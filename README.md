# Digital Design -- EITF65

This is the repo for the Quarto digitalteknik project, the *prepare-for-exam* guide.

## Usage
Download the latest release and open the `.html` file in your browser.

## Contributing
You are welcome to e-mail me. Other than that, for **big** issues, create an issue on GitHub.

If you want to build the project yourself, follow [Quarto Get Started Guide](https://quarto.org/docs/get-started/).

For it to build you also need to add [quarto-quizdown](https://github.com/parmsam/quarto-quizdown). Follow the guide on the linked page.
